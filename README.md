# Property Deal Scanner

A lightweight acquisition intelligence tool for identifying undervalued property opportunities.

Live dashboard:
https://property-deal-scanner.vercel.app

Features:
• Automated deal scoring
• £/sqft undervaluation detection
• Refurbishment opportunity modelling
• Rental yield estimation
• Price drop tracking
• Days-on-market analysis

API Endpoints:

/api/deals
Returns current deals dataset

/api/scan
Simulates fresh market scan

Tech stack:
• HTML dashboard
• Node serverless API
• Vercel hosting
• GitHub deployment pipeline
